1- style dentro de html-- <div class="content" style="background-color: pink" id="div"> 
Dentro de css: 
2- id --    #div.content {background-color: purple;}
3- clases --   div#div {background-color: grey;}
4- atributos --  #div {background-color: blue;}
5- pseudo clases --  div.content {background-color: yellow;}
6- selectores de tipo  --  .content {background-color: red;}
7- pseudo elementos --  div {background-color: black;}
